üîß Installation & Setup

System Requirements

  • Python: 3.7 or higher
  • Operating System: Windows 10+, macOS 10.14+, Ubuntu 18.04+
  • Memory: 4GB RAM minimum, 8GB recommended
  • Storage: 2GB free space
  • Graphics: DirectX 11 compatible (for hardware acceleration)

Quick Installation

# Clone the repository
git clone https://github.com/martin298298/tabulka.git
cd tabulka

# Install Python dependencies
pip install -r requirements.txt

# Install browser dependencies (choose one)
# Option 1: Playwright (recommended)
playwright install chromium

# Option 2: Selenium (fallback)
pip install selenium
# Download ChromeDriver from https://chromedriver.chromium.org/

# Optional: TTS dependencies for Linux
sudo apt-get install espeak espeak-data
sudo apt-get install python3-tk

Configuration Files

  • requirements.txt: Python package dependencies
  • config.yaml: System configuration (optional)
  • .env: Environment variables for credentials

Verification

# Test the installation
python enhanced_test.py

# Run basic functionality test
python test_complete_solution.py

# Start GUI application
python main_with_gui.py

üîå API Reference

Prediction API

RoulettePredictionSystem

Main class for roulette prediction functionality

from main import RoulettePredictionSystem

# Initialize the system
system = RoulettePredictionSystem(
    url="https://www.tokyo.cz/game/tomhornlive_56",
    headless=False,
    email="your_email@example.com",
    password="your_password"
)

# Start prediction
await system.initialize()
await system.run()

# Stop system
system.stop()
await system.cleanup()

Key Methods

  • initialize(): Initialize all system components
  • run(): Start the main prediction loop
  • process_frame(frame): Process a single video frame
  • stop(): Stop the prediction system
  • cleanup(): Clean up resources and show statistics

Text-to-Speech API

TTSSystem

Multi-voice text-to-speech system with language detection

from tts_system import get_tts_system

# Get TTS instance
tts = get_tts_system()

# Basic usage
tts.speak("Prediction: Number 17")

# Advanced settings
tts.set_voice("Microsoft Zira Desktop")
tts.set_rate(150)  # Words per minute
tts.set_volume(0.8)  # 0.0 to 1.0

# Prediction announcements
tts.speak_prediction("Number 17", confidence=0.85)

# Language detection and filtering
tts.set_filter_brackets(True)  # Filter [bracketed] content
tts.speak("Prediction [debug info] Number 17")  # Only "Prediction Number 17" spoken

Computer Vision API

RouletteVision

Advanced computer vision for wheel and ball detection

from vision import RouletteVision

# Initialize vision system
vision = RouletteVision()

# Detect wheel in frame
wheel_center, wheel_radius = vision.detect_wheel(frame)

# Detect ball position
ball_position = vision.detect_ball(frame)

# Create visualization

# Get detection confidence
confidence = vision.get_detection_confidence()

üõ†Ô∏è Troubleshooting

Common Issues

Installation Problems

ModuleNotFoundError: No module named 'playwright'
Solution: Install Playwright and browser dependencies 
pip install playwright
playwright install chromium
TTS not working on Linux
Solution: Install espeak and audio dependencies 
sudo apt-get install espeak espeak-data
sudo apt-get install pulseaudio

Runtime Errors

Low FPS or poor performance
Solutions:
  • Enable headless mode for better performance
  • Close other resource-intensive applications
  • Reduce video quality in browser settings
  • Use hardware acceleration if available
Wheel detection failing
Solutions:
  • Ensure good lighting conditions in stream
  • Adjust detection sensitivity parameters
  • Verify stream URL is accessible
  • Check for overlays blocking the wheel
